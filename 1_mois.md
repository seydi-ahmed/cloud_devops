# Mois 1 — Fondations Linux, Réseau & Culture DevOps

## Objectif du mois

Acquérir une maîtrise solide de Linux, des bases réseau et comprendre en profondeur la philosophie DevOps.

---

## Temps total

* 2h/jour × 30 jours ≈ 60 heures

---

## Semaine 1 — Linux Fondamental

* Installation Ubuntu / Debian (VM ou PC)
* Navigation système de fichiers
* Commandes essentielles : ls, cd, cp, mv, rm, cat, less, grep
* Gestion utilisateurs & permissions
* TP : créer des utilisateurs, configurer des droits

Livrable : fiche récap commandes Linux

---

## Semaine 2 — Linux Avancé

* Processus (ps, top, htop)
* Services systemd
* Cron & automatisation
* Bash scripting niveau débutant
* TP : script de sauvegarde automatique

Livrable : 3 scripts bash utiles

---

## Semaine 3 — Réseau

* TCP/IP
* Ports & protocoles
* DNS, HTTP/HTTPS
* Nginx & Apache
* TP : déployer un serveur web local

Livrable : serveur web fonctionnel

---

## Semaine 4 — Git & DevOps

* Git avancé (rebase, stash, flow)
* CI/CD concepts
* Métiers DevOps, SRE, Cloud Engineer
* Cycle de vie logiciel moderne

Livrable : mini pipeline simulé

---

## Résultat

✅ Tu maîtrises Linux, réseau et les bases DevOps
✅ Tu es prêt pour le Cloud

## Outils utilisés

* Ubuntu
* Git
* Nginx
